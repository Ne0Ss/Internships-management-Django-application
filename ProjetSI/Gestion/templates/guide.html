Bienvenue dans notre application !


Afin d'ajouter un stage veuillez suivre les étapes suivantes:


Etape 1: Ajouter un organisme d'accueil

- Se rendre dans la page Entreprises et ajouter l'entreprise qui va accuillir le ou les stagiaires et où le stage va se dérouler.

- Entrer les éventuels promoteurs de l'entreprise en question, en créant un jury si l'enployé en est fait partie sinon laisser la case vide.

- Entrer les sujets PFE que pourrai proposer l'organisme.


Etape 2: Ajouter le ou les encadrants

- Se rendre dans la page Encadrants via la barre de navigation ou le menu d'accueil et ajouter le ou les potenciels encadrants de l'école qui vont prendre en charge les stagiaires.

- Vous pouvez aussi preciser le jury dont il fait membre ou en créer un si il n'y a pas encore d'instance jury de créée.


Etape 3: Ajouter le ou les stagiaires

- Se rendre dans la page Stagiaires et ajouter les étudiants à enregistrer dans le stage.

- Les afilier à leur equipes respectives si elles sont déjà créées sinon en ajouter une.

- Assigner chaque équipe à son stage respectifs.

Note: tout les cas non désirés sont pris en compte par l'application afin d'assurer la bonne coherence de la base de données, par exemple, une équipe de 4 stagiaires ne peut pas être assignée à un stage de type stage ouvrier car ces stages n'acceptent que les monômes ou binômes.

Etape 4: Ajouter un stage

- Se rendre dans la page des Stages pour ajouter le stage en question.

- Vous aurai par la suite la possibilité d'entrer le rapport de stage, d'ajouter une fiche d'évaluation, et, si le stage est de type PFE, de choisir le sujet du stage parmis les sujets proposés par l'organisme d'accueil.

Note: tout les cas non désirés sont pris en compte par l'application afin d'assurer la bonne coherence de la base de données, par exemple, la durée d'un stage technique de courte durée doit etre comprise entre 6 et 8 semaines.
